# magneto
A simple magnetometer

## Parts
- FG-3 - fluxgate magnetometer (from fgsensors.com)
- Raspberry Pi Zero
- Raspberry Pico
- DS 18B20 tempersture sensor (on breakout board)
- Power supply components (7805, 78L05), etc.
